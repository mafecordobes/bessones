# Bessones Turismo Project

Project for a turism web page developing with **WordPress** CMS

## Requirements

-   PHP 7.3 or greater
-   MySQL 5.6 or greater Or MariaDB 10.0 or greater
-   Nginx or Apache with mod_rewrite
-    PHP Memory Limit 128MB or more

## Technologies

- WordPress
- jQuery 
- SASS
- HTML5 
- PHP
